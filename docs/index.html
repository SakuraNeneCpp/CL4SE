v1.0.0 · Windows / macOS / Linux

Enterで送信、 Caps LockでIME変換確定。

CLIMEは、Caps Lockキーの挙動を日本語IMEの変換確定で上書きする常駐ツールです。 変換確定のつもりでEnterを押し、そのままメッセージを送ってしまう事故を減らします。

すぐに使う ↓ v1.0.0をダウンロード ↗

無料・オープンソース 単一バイナリ フェイルセーフ

Why CLIME

誤送信を防ぎます

特に英語圏で開発されたアプリでは、しばしばメッセージの送信にEnterが割り当てられます。 しかし日本語ではIMEの変換確定にもEnterが割り当てられ、これが誤送信の原因になります。 そこでCLIMEは使われていないCaps LockにIME確定を任せることで、誤送信を防止します。

01

簡単無料インストール

このページの下の方にあるコマンドをコピペするだけで導入できます。アンインストールのコマンドもあるので、不満があれば簡単にアンインストールできます。

02

安心安全設計

IMEがオンで、変換中だと判断できる場合だけ確定キーを注入し、不明な状態では何もしないフェイルセーフ設計により、確実に誤送信を防止します。また、オープンソースなので不審なバイナリでないことが検証できます。

03

3つのOSで同じ使い心地

Windows・macOS・Linuxに対応。共通の判定ロジックで、環境が変わっても操作はCaps Lockひとつです。安心してタイピングのマッスルメモリーに組み込めます。

Fail-safe by design

迷ったら、押さない。

  • IMEオン + 変換中
    確定できると判断     COMMIT
  • IMEオン + 非変換中
    入力は確定済み     DO NOTHING
  • IMEオフ / 状態不明
    誤注入を避ける     DO NOTHING

Runtime

実行環境

GUIや常駐ランタイムを追加せず、各OS向けの単一バイナリとして動作します。

x86_64

Windows

  • 管理者権限不要
  • MS-IME / Google 日本語入力
  • ログイン時自動起動に対応
  • UACセキュアデスクトップは対象外
UNIVERSAL

macOS

  • Apple Silicon / Intel
  • 入力監視・アクセシビリティ許可が必要
  • 標準日本語IMなどに対応
  • Secure Keyboard Entry中は停止
x86_64

Linux

  • X11 / Wayland
  • fcitx5 / IBus
  • evdev + uinput
  • inputグループ・udev設定が必要

One-line setup

インストール

OSを選び、コマンドをターミナルにコピペするだけ。ダウンロード、診断、ログイン時の自動起動登録、今すぐの起動までまとめて完了します。

Windows x64

PowerShellで実行します。管理者として起動する必要はありません。

PowerShell
インストール + スタートアップ登録 + 起動 
$ErrorActionPreference="Stop"; $d="$env:LOCALAPPDATA\Programs\CLIME"; $e="$d\clime.exe"; New-Item -ItemType Directory -Force $d | Out-Null; if (-not (Test-Path $e)) { Invoke-WebRequest "https://github.com/SakuraNeneCpp/CLIME_CapsLockIMEcommit/releases/download/v1.0.0/clime-windows-x86_64.exe" -OutFile $e }; & $e doctor; if ($LASTEXITCODE -ne 0) { throw "CLIME doctor failed" }; & $e install-autostart; if ($LASTEXITCODE -ne 0) { throw "CLIME autostart setup failed" }; if (-not (Get-Process clime -ErrorAction SilentlyContinue)) { Start-Process -WindowStyle Hidden -FilePath $e -ArgumentList "run" }
アンインストール 
$d="$env:LOCALAPPDATA\Programs\CLIME"; $e="$d\clime.exe"; if (Test-Path $e) { & $e uninstall-autostart }; Get-Process clime -ErrorAction SilentlyContinue | Stop-Process; Remove-Item -LiteralPath $e -ErrorAction SilentlyContinue; Remove-Item -LiteralPath $d -ErrorAction SilentlyContinue

macOS universal

Apple Silicon / Intel共通です。初回だけTCC権限の手動許可が必要です。

Terminal
導入 — 貼り付けるのはこの一行だけ 
d="$HOME/Applications/CLIME"; e="$d/clime"; mkdir -p "$d" && { [ -x "$e" ] || { curl -fL "https://github.com/SakuraNeneCpp/CLIME_CapsLockIMEcommit/releases/download/v1.0.0/clime-macos-universal" -o "$e" && chmod 755 "$e"; }; } && { "$e" doctor >/dev/null 2>&1 || { "$e" run >/tmp/clime-setup.log 2>&1 & p=$!; sleep 3; kill -INT "$p" 2>/dev/null || true; wait "$p" 2>/dev/null || true; open "x-apple.systempreferences:com.apple.preference.security?Privacy"; }; } && "$e" install-autostart
アンインストール 
e="$HOME/Applications/CLIME/clime"; [ ! -x "$e" ] || "$e" uninstall-autostart; pkill -TERM -x clime 2>/dev/null || true; rm -f "$e"; rmdir "$HOME/Applications/CLIME" 2>/dev/null || true

システム設定が開いた場合だけ、「入力監視」と「アクセシビリティ」でCLIMEをオンにしてください。この確認だけはmacOSが手動操作を必須にしています。登録後は自動で起動します。

Linux x64

Wayland / X11、fcitx5 / IBusに対応します。最初にdoctorの案内で権限を整えます。

Shell
導入 — 貼り付けるのはこの一行だけ 
d="$HOME/.local/bin"; e="$d/clime"; s="$HOME/.local/share/clime"; r="/etc/udev/rules.d/99-clime-uinput.rules"; mkdir -p "$d" "$s" && { [ -x "$e" ] || { curl -fL "https://github.com/SakuraNeneCpp/CLIME_CapsLockIMEcommit/releases/download/v1.0.0/clime-linux-x86_64" -o "$e" && chmod 755 "$e"; }; } && { getent group input >/dev/null || sudo groupadd input; } && { id -nG "$USER" | tr ' ' '\n' | grep -qx input || { sudo usermod -aG input "$USER" && : >"$s/managed-input-group"; }; } && { [ -e /dev/uinput ] || sudo modprobe uinput; } && { [ -f "$r" ] || { printf '%s\n' 'KERNEL=="uinput", SUBSYSTEM=="misc", GROUP="input", MODE="0660", OPTIONS+="static_node=uinput"' | sudo tee "$r" >/dev/null && : >"$s/managed-uinput-rule"; }; } && sudo udevadm control --reload-rules && sudo udevadm trigger --name-match=uinput && "$e" install-autostart && { systemctl --user stop clime.service >/dev/null 2>&1 || true; } && mkdir -p "$HOME/.local/state" && if sg input -c "'$e' doctor"; then { pgrep -x clime >/dev/null || sg input -c "nohup '$e' run >'$HOME/.local/state/clime.log' 2>&1 &"; }; else "$e" uninstall-autostart; false; fi
アンインストール 
e="$HOME/.local/bin/clime"; s="$HOME/.local/share/clime"; r="/etc/udev/rules.d/99-clime-uinput.rules"; [ ! -x "$e" ] || "$e" uninstall-autostart; pkill -TERM -x clime 2>/dev/null || true; rm -f "$e"; if [ -f "$s/managed-uinput-rule" ]; then sudo rm -f "$r" && sudo udevadm control --reload-rules && sudo udevadm trigger --name-match=uinput; rm -f "$s/managed-uinput-rule"; fi; if [ -f "$s/managed-input-group" ]; then sudo gpasswd -d "$USER" input >/dev/null 2>&1 || true; rm -f "$s/managed-input-group"; fi; rmdir "$s" 2>/dev/null || true

途中でsudoパスワードを求められたら入力してください。inputグループとuinputルールも自動設定し、現在はすぐ起動、次回ログイン以降は自動起動します。アンインストール時は、この一行が追加した権限設定も戻します。
